# Usage

Run IC/DC by double clicking run.bat (Windows), or running `python -m icdc` in the console (Mac/Linux).

# Dependencies

- Python 2.7, NumPy, Scipy
- PyQt/PySide, pyqtgraph
- pyopengl
- PyDICOM
- PyWavelets
- sklearn
